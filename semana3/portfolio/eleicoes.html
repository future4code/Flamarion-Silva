Projeto Eleções Tenológica

Este é um projeto muito interessante. O qual foi feito com o intuito de práticar e aprimorar meus skills e meu conhecimento em javascript, CSS e HTML. Enfim, consiste num página à qual possui 4 grandes nomes da tecnologia, e você precisa voltar em um deles para ser o novo presidente; kkkkkkkk. POis é, a parte criativa são os calculos elabadorados em javascript para calcular o resultado.
